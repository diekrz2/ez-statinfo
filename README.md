# ez-statinfo
Easily view CPU, RAM, and disk usage

Allows you to simply and rudimentary display the amount of CPU, RAM, and disk used. A small window with a black background and white text. Drag while holding down the left mouse button on the white line. Tested on Bodhi Linux 7 and Linux Mint 22

**Install:**
------------
Download the ez-statinfo folder, navigate to the folder in the terminal, and run the 'install.sh' script with '_bash install.sh_'. The '_Easy System Info_' entry will be added to the main menu. 

**Dependencies:** python3-psutil and python3-tk.
